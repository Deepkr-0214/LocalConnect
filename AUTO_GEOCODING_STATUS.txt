HOW IT WORKS
───────────────────────────────────────────────────────────────────────────────

WHEN VENDOR REGISTERS:
  1. Vendor enters: Q.no-57/21, Jamshedpur, Jharkhand
  2. System calls: geocode_address(business_address)
  3. API returns: latitude=22.8015, longitude=86.2029
  4. Database saves: vendor.latitude, vendor.longitude
  5. User sees: "✅ Location detected automatically!"
  6. Result: Vendor appears on map at correct location


WHEN VENDOR UPDATES ADDRESS:
  1. Vendor changes: Address field in settings
  2. System detects: address_changed = True
  3. System calls: geocode_address(new_address)
  4. Database updates: new latitude/longitude
  5. User sees: "Location updated automatically!"
  6. Result: Map location updates


WHEN CUSTOMER UPDATES ADDRESS:
  1. Customer enters: Address, City, State, Pincode
  2. System calls: geocode_address(full_address)
  3. Database saves: customer.latitude, customer.longitude
  4. Result: Distance-based vendor filtering works
